:target-within

This feature is discouraged. Consider using :has() , :target instead. For the rationale, see:

  • https://drafts.csswg.org/selectors-4/#the-target-pseudo
  • https://github.com/w3c/csswg-drafts/issues/8357

The :target-within CSS pseudo-class matches the element with an ID matching the URL fragment and its ancestors.
