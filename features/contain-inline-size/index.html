Inline-size containment

The contain: inline-size CSS declaration prevents the element's inline dimension from being set by the element's contents. This permits the browser to avoid slower layout calculations.

Status

Baseline Widely Available (since 2025-03-12)

MDN documentation

  • Inline-size, in contain

Specifications

  • CSS Containment Module Level 3 (#containment-inline-size), from Cascading Style Sheets (CSS) Working Group (W3C).
