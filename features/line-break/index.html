line-break

The line-break CSS property sets how strictly to apply rules for wrapping text to new lines, especially for symbols and punctuation.

Status

Baseline Widely Available (since 2023-01-28)

MDN documentation

  • line-break

Specifications

  • CSS Text Module Level 4 (#line-break-property), from Cascading Style Sheets (CSS) Working Group (W3C).
