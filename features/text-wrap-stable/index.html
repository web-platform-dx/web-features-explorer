text-wrap: stable

The text-wrap: stable CSS declaration wraps text using the default algorithm, but does not change wrapping on earlier lines while the text is being edited.

Status

Baseline Newly Available (since 2024-10-17)

This feature is expected to reach Baseline Widely Available status on: 2027-04-17

MDN documentation

  • Stable, in text-wrap

Specifications

  • CSS Text Module Level 4 (#valdef-text-wrap-style-stable), from Cascading Style Sheets (CSS) Working Group (W3C).
