HWB

The hwb() CSS function picks colors using hue, whiteness, and blackness channels.

Status

Baseline Widely Available (since 2024-10-28)

MDN documentation

  • hwb()

Specifications

  • CSS Color Module Level 4 (#the-hwb-notation), from Cascading Style Sheets (CSS) Working Group (W3C).
