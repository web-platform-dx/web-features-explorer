Hyphenate character

Newly available since 2023-09-18
Tags: css Groups: CSS > Text > Text wrap

The hyphenate-character CSS property sets the character or string to use at the end of a line before a line break.

Learn more

  • hyphenate-character

Specifications

  • CSS Text Module Level 4 (#hyphenate-character), from Cascading Style Sheets (CSS) Working Group (W3C).
