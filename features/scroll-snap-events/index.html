Scroll snap events

Limited availability
Tags: api Groups: Scrolling

The scrollsnapchanging and scrollsnapchange events fire when a scroll gesture changes the selected scroll snap target. To learn more, see Document: scrollsnapchange event, Document: scrollsnapchanging event, Element: scrollsnapchange event, Element: scrollsnapchanging event, SnapEvent, SnapEvent: SnapEvent() constructor, SnapEvent: snapTargetBlock property, SnapEvent: snapTargetInline property, Window: scrollsnapchange event, Window: scrollsnapchanging event at MDN.
