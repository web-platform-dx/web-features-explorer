Performance

The performance global object and the PerformanceObserver API provide access to performance-related information for the current execution context.

Status

Baseline Widely Available (since 2018-03-16)

Specifications

  • High Resolution Time, from Web Performance Working Group (W3C).
  • Performance Timeline, from Web Performance Working Group (W3C).
  • User Timing (#extensions-performance-interface), from Web Performance Working Group (W3C).
