document.write()

This feature is discouraged — The dynamic markup insertion API, which includes document.write(), has "very idiosyncratic behavior" which can be difficult to debug.

Consider using DOM instead.

For more information, see: dynamic-markup-insertion.html#document.write() (HTML spec)

The document.open(), document.write(), document.writeln(), and document.close() methods write content to a document, possibly while that document is still loading.
