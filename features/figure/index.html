<figure> and <figcaption>

Widely available since 2018-01-29
Tags: html Groups: HTML > HTML elements

The <figure> and <figcaption> HTML elements represent an illustration, diagram, text, or other self-contained reference content, with an optional caption.

Learning resources on MDN

  • <figure>: The Figure with Optional Caption element
  • <figcaption>: The Figure Caption element
