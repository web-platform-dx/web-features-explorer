Array iterators

Arrays are iterable with the for … of statement and enumerable with the methods entries(), keys(), and values().

Status

Baseline Widely Available (since 2020-11-09)

MDN documentation

  • for...of

Specifications

  • ECMAScript® 2026 Language Specification (#sec-array.prototype-@@iterator), from TC39 (Ecma International).
  • ECMAScript® 2026 Language Specification (#sec-array.prototype.entries), from TC39 (Ecma International).
  • ECMAScript® 2026 Language Specification (#sec-array.prototype.keys), from TC39 (Ecma International).
  • ECMAScript® 2026 Language Specification (#sec-array.prototype.values), from TC39 (Ecma International).
