HTML wrapper methods

This feature is discouraged — HTML wrapper methods exist only for backwards compatibility. Use DOM APIs to create elements instead.

Consider using DOM instead.

For more information, see: https://tc39.es/ecma262/multipage/additional-ecmascript-features-for-web-browsers.html#sec-additional-ecmascript-features-for-web-browsers

JavaScript built-in methods such as, String.bold() and String.italics(), wrap strings in (often historic) HTML.
