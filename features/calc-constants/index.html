calc() keywords

The e, pi, infinity, and NaN keywords represent well-defined constants accepted in CSS math functions such as calc().

Status

Baseline Newly Available (since 2023-06-06)

This feature is expected to reach Baseline Widely Available status on: 2025-12-06

MDN documentation

  • <calc-keyword>

Specifications

  • CSS Values and Units Module Level 4 (#calc-keywords), from Cascading Style Sheets (CSS) Working Group (W3C).
