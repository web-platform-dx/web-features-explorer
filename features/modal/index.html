:modal

The :modal pseudo-class selects elements that put other elements into a non-interactive state, such as a fullscreen element or a <dialog> element used with showModal().

Status

Baseline Widely Available (since 2025-03-02)

MDN documentation

Specifications

  • Selectors Level 4 (#modal-state), from Cascading Style Sheets (CSS) Working Group (W3C).
