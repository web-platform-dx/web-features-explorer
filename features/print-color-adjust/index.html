print-color-adjust

Newly available since 2025-05-01
Tags: css Groups: Print media

The print-color-adjust CSS property sets whether styles of printed pages should be adjusted to use less ink, in cases such as light text on a dark background.

Learn more

  • print-color-adjust

Specifications

  • CSS Color Adjustment Module Level 1 (#propdef-print-color-adjust), from Cascading Style Sheets (CSS) Working Group (W3C).
