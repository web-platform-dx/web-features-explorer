::placeholder

The ::placeholder CSS pseudo-element selects help text in <input> and <textarea> elements when no value is set.

Status

Baseline Widely Available (since 2022-07-15)

MDN documentation

  • ::placeholder

Specifications

  • CSS Pseudo-Elements Module Level 4 (#placeholder-pseudo), from Cascading Style Sheets (CSS) Working Group (W3C).
