overflow: overlay

This feature is discouraged — Some old browsers may implement non-standard behavior for overflow: overlay.

Consider using overflow instead.

For more information, see: css-overflow-3/#valdef-overflow-overlay (CSSWG draft)

The overflow: overlay CSS declaration is an alias to overflow: auto. Historically, it caused non-standard behavior, allowing scrollbars to overlay content without taking up layout space.

Status

Discouraged
