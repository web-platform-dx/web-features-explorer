calc()

The calc() CSS function computes mathematical expressions such a calc(100%/3 - 1em).

Status

Baseline Widely Available (since 2018-01-29)

MDN documentation

  • calc()

Specifications

  • CSS Values and Units Module Level 3 (#calc-notation), from Cascading Style Sheets (CSS) Working Group (W3C).
