Rhythmic sizing

Limited availability
Tags: css Groups: CSS > Text

The line-height-step CSS property creates consistent vertical rhythm by setting the spacing between body text baselines. It rounds taller lines like headings up to the next multiple of the body text spacing. To learn more, see line-height-step at MDN.
