RGB

The rgb(), rgba(), and hexadecimal (as in #004488ff) notations pick colors using red, green, blue, and alpha (transparency) channels.

Status

Baseline Widely Available (since 2022-07-15)

MDN documentation

  • rgb()
  • <hex-color>

Specifications

  • CSS Color Module Level 4 (#rgb-functions), from Cascading Style Sheets (CSS) Working Group (W3C).
