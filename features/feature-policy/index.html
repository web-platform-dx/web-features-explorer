Feature policy

This feature is discouraged — Feature policy is superseded by permissions policy.

Consider using Permissions policy instead.

For more information, see: w3c/webappsec-permissions-policy/pull/379 (GitHub)

The Feature-Policy response header sets whether a policy-controlled feature, such as an API, may be used in a document. Not to be confused with permissions policy.
