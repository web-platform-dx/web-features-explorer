<input type="hidden">

The <input type="hidden"> HTML element represents a form field that is not shown visually but is still included in the form submission.

Status

Baseline Widely Available (since 2018-01-29)

Specifications

  • HTML (#hidden-state-(type=hidden)), from HTML Workstream (WHATWG).
