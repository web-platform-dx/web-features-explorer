::spelling-error and ::grammar-error

The ::spelling-error and ::grammar-error CSS pseudo-elements match text that is highlighted as misspelled and grammatically incorrect, respectively.

Status

Limited availability

MDN documentation

  • ::grammar-error
  • ::spelling-error

Specifications

  • CSS Pseudo-Elements Module Level 4 (#selectordef-spelling-error), from Cascading Style Sheets (CSS) Working Group (W3C).
  • CSS Pseudo-Elements Module Level 4 (#selectordef-grammar-error), from Cascading Style Sheets (CSS) Working Group (W3C).
