Resource timing (initial support)

Widely available since 2020-03-19
Tags: api Groups: Performance

PerformanceResourceTiming entries report when network events happen while loading a resource, such as when connections start and end. You can use this information to measure loading times.

Learn more

  • PerformanceResourceTiming: decodedBodySize property
  • PerformanceResourceTiming: encodedBodySize property
  • PerformanceResourceTiming: transferSize property

Specifications
