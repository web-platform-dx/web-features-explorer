::backdrop

Widely available
Tags: css Groups: CSS > Selectors

The ::backdrop CSS pseudo-element is a box underneath an element in the top layer, such as a <dialog>. It can be used to create obscuring effects distinguishing a modal element from the elements underneath. To learn more, see ::backdrop at MDN.
