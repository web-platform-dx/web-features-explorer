Optional catch binding

Omit the the binding parameter of a catch clause when you don't need information about the exception in a try ... catch statement.

Status

Baseline Widely Available (since 2022-07-15)

MDN documentation

  • Catch_binding, in try...catch

Specifications

  • ECMAScript® 2026 Language Specification (#sec-try-statement), from TC39 (Ecma International).
