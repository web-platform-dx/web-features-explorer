PerformanceTiming and PerformanceNavigation

This feature is discouraged. Consider using Performance , Navigation timing instead. For the rationale, see:

  • https://w3c.github.io/navigation-timing/#obsolete

The window.performance.timing and window.performance.navigation APIs report performance timing information for events that occur during the loading and use of the current page.

Status

Discouraged

MDN documentation

  • Performance: timing property
  • Performance: navigation property

Specifications

  • Navigation Timing Level 2 (#obsolete), from Web Performance Working Group (W3C).
