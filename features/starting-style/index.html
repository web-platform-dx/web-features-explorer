@starting-style

Newly available since 2024-08-06
Tags: api css Groups: CSS

The @starting-style CSS at-rule sets the starting values for properties that are transitioning when the target element's style is first updated.

Learn more

  • @starting-style

Specifications

  • CSS Transitions Level 2 (#defining-before-change-style), from Cascading Style Sheets (CSS) Working Group (W3C).
