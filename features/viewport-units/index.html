Viewport units

The vw, vh, vmin, and vmax CSS viewport units are relative to the size of the viewport, and are used to size elements in relation to the viewport's dimensions.

Status

Baseline Widely Available (since 2020-04-17)

MDN documentation

  • Relative length units based on viewport, in <length>

Specifications

  • CSS Values and Units Module Level 4 (#viewport-relative-lengths), from Cascading Style Sheets (CSS) Working Group (W3C).
