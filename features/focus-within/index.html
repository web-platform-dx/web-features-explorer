:focus-within

Widely available since 2022-07-15
Tags: css Groups: CSS > Selectors

The :focus-within CSS pseudo-class matches an element if the element or any of its children are focused.

Learn more

  • :focus-within

Specifications

  • Selectors Level 4 (#the-focus-within-pseudo), from Cascading Style Sheets (CSS) Working Group (W3C).
