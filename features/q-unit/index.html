Q unit

The Q CSS length unit is an absolute length anchored to the physical measurement of quarter-millimeters. 1Q is equivalent to 1/40 of 1 centimeter.

Status

Baseline Widely Available (since 2022-09-24)

MDN documentation

  • Q, in <length>

Specifications

  • CSS Values and Units Module Level 4 (#absolute-lengths), from Cascading Style Sheets (CSS) Working Group (W3C).
