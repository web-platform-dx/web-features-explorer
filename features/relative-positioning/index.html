Relative positioning

Widely available since 2018-01-29
Tags: css Groups: CSS > Layout > Positioning

The position: relative CSS declaration offsets the position of an element relative to its position in the normal flow.

Learn more

  • Relative, in position

Specifications

  • CSS Positioned Layout Module Level 3 (#relpos-insets), from Cascading Style Sheets (CSS) Working Group (W3C).
