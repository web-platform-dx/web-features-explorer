display: flow-root

The display: flow-root CSS declaration sets an element as the root element of a new flow layout for its children, preventing margin collapse with sibling elements.

Status

Baseline Widely Available (since 2022-07-15)

MDN documentation

  • Flow-root, in display

Specifications

  • CSS Display Module Level 3 (#inner-model), from Cascading Style Sheets (CSS) Working Group (W3C).
