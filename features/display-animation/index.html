display animation

You can animate elements between display: none and any other display value or animate between content-visibility: hidden and any other content-visibility value. This also applies to transitions.

Status

Limited availability

MDN documentation

  • Animating_display, in display
  • Animating_and_transitioning_content-visibility, in content-visibility

Specifications

  • CSS Display Module Level 4 (#display-animation), from Cascading Style Sheets (CSS) Working Group (W3C).
  • CSS Containment Module Level 3 (#content-visibility-animation), from Cascading Style Sheets (CSS) Working Group (W3C).
