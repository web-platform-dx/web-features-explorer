Table styling

This feature is discouraged — Some table styling attributes have no effect. They are superseded by CSS.

For more information, see: obsolete.html (HTML spec)

Attributes and properties of table elements, such as align, vAlign, noWrap, bgcolor, or char set various visual presentation properties of the table and its sections and cells.

Status

Discouraged
