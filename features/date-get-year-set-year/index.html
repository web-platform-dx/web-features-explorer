getYear() and setYear()

Discouraged
Tags: javascript Groups: JavaScript

This feature is discouraged. For the rationale, see:

  • https://tc39.es/ecma262/multipage/additional-ecmascript-features-for-web-browsers.html#sec-additional-ecmascript-features-for-web-browsers

The getYear() and setYear() methods of Date objects get and set the year as an offset to 1900.

Learn more

  • Date.prototype.setYear()

Specifications
