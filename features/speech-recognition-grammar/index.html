Speech recognition grammar

This feature is discouraged. For the rationale, see:

  • https://github.com/WebAudio/web-speech-api/pull/117

The SpeechRecognition.grammars property of the Web Speech API represents a set of words or patterns of words that the recognition service should recognize, used for when the recognition service's accuracy is low. This feature has been removed due to advancements in speech recognition.

Status

Discouraged
