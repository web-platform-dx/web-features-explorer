align-content in block layouts

The align-content property vertically aligns content in block layouts, like it does in flex or grid layouts.

Status

Baseline Newly Available (since 2024-04-16)

This feature is expected to reach Baseline Widely Available status on: 2026-10-16

MDN documentation

Specifications

  • CSS Box Alignment Module Level 3 (#align-justify-content), from Cascading Style Sheets (CSS) Working Group (W3C).
