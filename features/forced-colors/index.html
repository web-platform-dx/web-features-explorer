Forced colors

Newly available since 2022-09-12
Tags: css Groups: CSS

The forced-colors CSS @media rule detects when a user has chosen to use a forced colors mode, also known as high-contrast mode, and the forced-color-adjust CSS property sets whether forced colors apply to an element. To learn more, see forced-colors, forced-color-adjust at MDN.
