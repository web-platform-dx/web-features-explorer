font-optical-sizing

The font-optical-sizing CSS property sets whether text rendering is optimized for viewing at different sizes.

Status

Baseline Widely Available (since 2022-09-24)

MDN documentation

  • font-optical-sizing

Specifications

  • CSS Fonts Module Level 4 (#font-optical-sizing-def), from Cascading Style Sheets (CSS) Working Group (W3C).
