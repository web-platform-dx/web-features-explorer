Web authentication

Widely available since 2024-03-07
Tags: api html http

The web authentication API allows you to create public key-based credentials and use them for authentication, such as when signing in. Also known as WebAuthn.

Learn more

Specifications

  • Web Authentication: An API for accessing Public Key Credentials - Level 3, from Web Authentication Working Group (W3C).
