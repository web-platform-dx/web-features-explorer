ric unit

The ric CSS length unit, or root international character, is a font-relative length equal to the width of CJK character relative to the root element.

Status

Limited availability

MDN documentation

Specifications

  • CSS Values and Units Module Level 4 (#font-relative-lengths), from Cascading Style Sheets (CSS) Working Group (W3C).
