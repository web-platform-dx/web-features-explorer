font-synthesis-style

The font-synthesis-style CSS property sets whether or not the browser should synthesize italic and oblique typefaces when they're missing from the font.

Status

Baseline Widely Available (since 2025-09-27)

Specifications

  • CSS Fonts Module Level 4 (#font-synthesis-style), from Cascading Style Sheets (CSS) Working Group (W3C).
