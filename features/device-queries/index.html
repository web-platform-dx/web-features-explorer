Device media queries

This feature is discouraged — Device media queries do not reflect available layout, only the physical size of the device.

Consider using Media queries instead.

For more information, see: mediaqueries-5/#mf-deprecated (CSSWG draft)

The device-aspect-ratio, device-height, and device-width media features describe physical properties of the displaying output device.
