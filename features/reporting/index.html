Reporting API

The Reporting-Endpoints HTTP header and ReportingObserver() API send reports of Content Security Policy (CSP) violations, Permissions-Policy violations, deprecated feature usage, browser interventions, and crashes to a nominated URL or callback function.

Status

Limited availability

  • Content Security Policy Level 3 (#reporting), from Web Application Security Working Group (W3C).
