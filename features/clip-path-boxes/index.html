Clip path boxes

Newly available since 2023-11-02
Tags: css Groups: Clipping, shapes and masking

The fill-box, stroke-box, and view-box values for clip-path set an edge of the element's box to use as the clipping shape.

Learn more

  • Geometry-box, in clip-path

Specifications

  • CSS Masking Module Level 1 (#the-clip-path), from Cascading Style Sheets (CSS) Working Group (W3C).
