Private network access

This feature is discouraged — Private network access caused compatibility problems and is superseded by local network access.

Consider using Local network access instead.

For more information, see: WICG/local-network-access?tab=readme-ov-file#local-network-access (GitHub) , https://developer.chrome.com/blog/pna-on-hold

The Access-Control-Request-Private-Network: true and Access-Control-Allow-Private-Network: true HTTP request and response headers establish permission for websites to make requests to internal networks, such as to routers.
