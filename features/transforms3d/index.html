3D transforms

The transform CSS property and its 3D transform functions allow rotations and other transforms in three dimensions, including perspective transforms.

Status

Baseline Widely Available (since 2024-09-14)

Specifications

  • CSS Transforms Module Level 2, from Cascading Style Sheets (CSS) Working Group (W3C).
