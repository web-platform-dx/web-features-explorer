RegExp compile()

This feature is discouraged — The compile() method duplicates behavior of the RegExp() constructor and it exists only for backwards compatibility.

For more information, see: https://tc39.es/ecma262/multipage/additional-ecmascript-features-for-web-browsers.html#sec-additional-ecmascript-features-for-web-browsers

The compile() method of RegExp objects recompiles an existing regular expression object using a new pattern and flags.
