SVG 1.1

This feature is discouraged — SVG 2 omittted some parts of SVG 1.1 to "improv[e] the usability and precision of the language."

Consider using SVG , @font-face instead.

For more information, see: https://svgwg.org/svg2-draft/changes.html#substantial

The SVG 1.1 image format has several components that were excluded from SVG 2, such as fonts, alternate glyphs, and the xlink namespace.

Status
