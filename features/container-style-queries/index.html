Container style queries

Container style queries with the @container at-rule apply styles to an element based on the values of custom properties of its container.

Status

Limited availability

MDN documentation

  • @container

Specifications

  • CSS Conditional Rules Module Level 5 (#style-container), from Cascading Style Sheets (CSS) Working Group (W3C).
