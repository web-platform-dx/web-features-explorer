User action pseudo-classes

The :active, :focus, and :hover CSS pseudo-classes match elements based on how users are interacting with them.

Status

Baseline Widely Available (since 2018-01-29)

MDN documentation

  • :active
  • :focus
  • :hover

Specifications

  • Selectors Level 4 (#the-hover-pseudo), from Cascading Style Sheets (CSS) Working Group (W3C).
  • Selectors Level 4 (#the-active-pseudo), from Cascading Style Sheets (CSS) Working Group (W3C).
  • Selectors Level 4 (#the-focus-pseudo), from Cascading Style Sheets (CSS) Working Group (W3C).
