Width and height

The width and height CSS properties set the preferred physical size of an element.

Status

Baseline Widely Available (since 2018-01-29)

Specifications

  • CSS Box Sizing Module Level 3 (#preferred-size-properties), from Cascading Style Sheets (CSS) Working Group (W3C).
