Web Cryptography

The Web Cryptography API performs low-level cryptographic operations, such as encryption, decryption, and signature verification. Also known as the Web Crypto API.

Status

Baseline Widely Available (since 2018-01-29)

MDN documentation

  • Web Crypto API

Specifications

  • Web Cryptography Level 2, from Web Application Security Working Group (W3C).
  • Secure Curves in the Web Cryptography API, from Web Platform Incubator Community Group (W3C).
