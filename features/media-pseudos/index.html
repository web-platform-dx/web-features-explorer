Media element pseudo-classes

The :playing, :paused, :seeking, :buffering, :stalled, :muted, and :volume-locked CSS pseudo-classes match <audio> and <video> elements based on their state.

Status

Limited availability

Specifications

  • Selectors Level 4 (#resource-pseudos), from Cascading Style Sheets (CSS) Working Group (W3C).
  • HTML (#pseudo-classes), from HTML Workstream (WHATWG).
