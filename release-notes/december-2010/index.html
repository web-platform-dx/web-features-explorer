New in Chrome

The following features are now available in Chrome:

  • <figure> and <figcaption>: The <figure> and <figcaption> HTML elements represent an illustration, diagram, text, or other self-contained reference content, with an optional caption.
  • <link rel="prefetch">: The rel="prefetch" attribute for the <link> HTML element is a hint to the browser that the user is likely to navigate to a resource, so the browser should preemptively fetch and cache the resource.
